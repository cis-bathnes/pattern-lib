BATHNES Digital Toolkit

This toolkit is a highly-modular design system for rapid web page development. It contains different materials that can be assembled into more complex page layouts.

This guide contains real working examples, code snippets, documentation, and style guidelines.

It's purpose is to enable internal and external developers to create high quality experiences more easily.

Usage

The toolkit is based around the following elements:

  • Styles: The core parts of the design and brand; colour, logos, layouts, typograpy
  • Components: Groups of elements that can be used in a design
  • Patterns: Design solutions to common problems made up of Components
  • Templates: Example page templates
